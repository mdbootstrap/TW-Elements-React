Properties

TEDropdown

Name Type Default Description
animation Boolean true Turn off/on animation to TEDropdown element.
autoClose String | Boolean true

Configure the auto close behavior of the dropdown:

  • true - clicking outside or inside the dropdown menu.
  • false - clicking the toggle button or manually toggling TEDropdown element using isOpen property. Also will not be closed by pressing esc key.
  • 'inside' - clicking only inside the dropdown menu.
  • 'outside' - clicking only outside the dropdown menu.
isOpen Boolean false Toggle visibility of the dropdown element.
tag String 'div' Defines tag of the TEDropdown element.
theme Object {} Allows to change the Tailwind classes used in the component.

TEDropdownToggle

Name Type Default Description
tag String 'button' Defines tag of the TEDropdownToggle element.

TEDrodpdownMenu

Name Type Default Description
alignment "start" | "end" "start" Set alignment of the TEDropdownMenu element.
appendToBody Boolean false Append TEDropdownMenu element to the body.
display "dynamic" | "static" "dynamic" By default, we use Popper for dynamic positioning. Disable this with static.
offset array | function [0, 0]

Offset of the dropdown relative to its target.

When a function is used to determine the offset, it is called with an object containing the offset data as its first argument. The function must return an object with the same structure. The triggering element DOM node is passed as the second argument.

For more information refer to Popper's offset docs.

popperConfig Object {} Set default popper options to dropdown element
position "dropup" | "dropright" | "dropleft" - Set position of the TEDropdownMenu element. Available valuse are dropup, dropleft, dropright. When is not set, the menu will be positioned to the bottom.
responsive "sm-start" | "md-start" | "lg-start" | "xl-start" | "xxl-start" | "sm-end" | "md-end" | "lg-end" | "xl-end" | "xxl-end" - Defines the responsive alignment of the TEDropdownMenu component.
tag String 'ul' Defines tag of the TEDropdownMenu element.
theme Object {} Allows to change the Tailwind classes used in the component.

TEDropdownItem

Name Type Default Description
preventCloseOnClick Boolean false Prevents dropdown from closing when clicked inside the TEDropdownItem element.
tag String 'li' Defines tag of the TEDropdownItem element.

Classes

Custom classes can be passed via theme prop. Create an object with classes like below and pass it to the prop.

TEDropdown

Name Default Description
dropdown relative Sets styles to the TEDropdown element.

TEDropdownMenu

Name Default Description
fade transition-opacity duration-300 ease-linear motion-reduce:transition-none Sets transition styles to the TEDropdownMenu element.
menu absolute z-[1000] float-left m-0.5 min-w-max list-none overflow-hidden rounded-lg border-none bg-white bg-clip-padding text-left text-base shadow-lg dark:bg-neutral-700 Sets styles to the TEDropdownMenu element.

Events

Event type Description
onHide Fires immediately when the onHide instance method has been called. It can be triggered in TEDropdown element.
onHidden Fired when the dropdown has finished being hidden from the user and CSS transitions have completed. It can be triggered in TEDropdown element.
onShow Fires immediately when the onShow instance method is called. It can be triggered in TEDropdown element.
onShown Fired when the dropdown has been made visible to the user and CSS transitions have completed. It can be triggered in TEDropdown element.
