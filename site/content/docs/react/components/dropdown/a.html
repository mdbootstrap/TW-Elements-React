Properties

TEDropdown

Name Type Default Description
animation Boolean true Turn off/on animation to TEDropdown element.
isOpen Boolean false Set open dropdown on mounted in TEDropdown element
options Object {} Set default popper options to TEDropdown element
tag String 'div' Defines tag of the TEDropdown element.
theme Object {} Allows to change the Tailwind classes used in the component.

TEDropdownToggle

Name Type Default Description
tag String 'button' Defines tag of the TEDropdownToggle element.

TEDrodpdownMenu

Name Type Default Description
alignment String - Set alignment of the TEDropdownMenu element. Available value is end. When is not set, the menu will be aligned to the left.
alwaysOpen Boolean false Sets TEDropdownMenu element to be always open.
appendToBody Boolean false Append TEDropdownMenu element to the body.
position String - Set position of the TEDropdownMenu element. Available valuse are dropup, dropleft, dropright. When is not set, the menu will be positioned to the bottom.
responsive String - Defines the responsive alignment of the TEDropdownMenu component. Available values are "sm-start" | "md-start" | "lg-start" | "xl-start" | "2xl-start" | "sm-end" | "md-end" | "lg-end" | "xl-end" | "2xl-end".
tag String 'ul' Defines tag of the TEDropdownMenu element.
theme Object {} Allows to change the Tailwind classes used in the component.

TEDropdownItem

Name Type Default Description
childTag String - Defines tag of the TEDropdownItem children element.
disabled Boolean false Disable TEDropdownItem element.
divider Boolean false Renders TEDropdownItem as a divider.
header Boolean false Renders TEDropdownItem as a header.
href String '#' Defines href in TEDropdownItem when it's rendered as a link.
link Boolean false Renders TEDropdownItem as a link.
linkClasses String - Defines classes of the TEDropdownItem link element.
tag String 'li' Defines tag of the TEDropdownItem element.
theme Object {} Allows to change the Tailwind classes used in the component.

Classes

Custom classes can be passed via theme prop. Create an object with classes like below and pass it to the prop.

TEDropdown

Name Default Description
dropdown relative Sets styles to the TEDropdown element.

TEDropdownMenu

Name Default Description
animationIn animate-[fade-in_0.31s_ease-in] Sets animation in styles to the TEDropdownMenu element.
animationOut animate-[fade-out_0.31s_ease-out] Sets animation out styles to the TEDropdownMenu element.
menu absolute z-[1000] float-left m-0.5 min-w-max list-none overflow-hidden rounded-lg border-none bg-white bg-clip-padding text-left text-base shadow-lg dark:bg-neutral-700 Sets styles to the TEDropdownMenu element.

TEDropdownItem

Name Default Description

divider my-2 h-0 border border-t-0 border-solid border-neutral-700 opacity-25 dark:border-neutral-200 Sets styles to the TEDropdownItem element when it's rendered as a divider.
header block w-full whitespace-nowrap pointer-events-none cursor-text bg-transparent px-4 py-2 text-sm text-left font-semibold text-neutral-500 dark:text-neutral-200 Sets styles to the TEDropdownItem element when it's rendered as a header.
item block w-full min-w-[160px] whitespace-nowrap bg-transparent px-4 py-2 text-sm text-left font-normal pointer-events-auto text-neutral-700 hover:bg-neutral-100 active:text-neutral-800 active:bg-neutral-100 focus:bg-neutral-100 focus:text-neutral-800 focus:outline-none active:no-underline dark:text-neutral-200 dark:hover:bg-neutral-600 dark:focus:bg-neutral-600 dark:active:bg-neutral-600 Sets default styles to the TEDropdownItem element.

Events

Event type Description
onHide Fires immediately when the onHide instance method has been called. It can be triggered in TEDropdown element.
onShow Fires immediately when the onShow instance method is called. It can be triggered in TEDropdown element.
