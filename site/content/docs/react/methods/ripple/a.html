Properties

TERipple

Name Type Default Description
rippleTag string "div" Defines tag of the TERipple element
rippleCentered boolean - Centers the position of ripple
rippleColor string 'dark' Changes color of ripple
rippleColorDark string - If present, changes the color of ripple in dark mode
rippleDuration number - Sets duration of animation
rippleRadius number - Sets radius value
rippleUnbound boolean false Sets whether the effect should go beyond the wrapper's boundaries
theme Object {} Allows to change the Tailwind classes used in the component

Classes

Custom classes can be passed via theme prop. Create an object with classes like below and pass it to the prop.

Name Default Description
ripple relative inline-block align-bottom Adds classes to the ripple wrapper.
rippleOverflow absolute left-0 top-0 overflow-hidden w-full h-full Sets classes the the ripple wave wrapper wave if the effect shouldn't go beyond the wrapper's boundaries.
unbound overflow-visible Allows the ripple wave to go beyond the wrapper's boundaries. Used in a ripple wave wrapper.
rippleWave block rounded-[50%] pointer-events-none absolute touch-none transition-[transform,_opacity] ease-[cubic-bezier(0,0,0.15,1),_cubic-bezier(0,0,0.15,1)] z-[999] scale-0 Sets visual styles of the ripple effect
rippleWaveActive scale-100 opacity-0 Sets styles when the ripple effect is active.
rippleWaveInactive scale-0 opacity-50 Sets styles when the ripple effect is inactive (the starting point)
